== VPStreamer
Copyright (c) Vidpresso Inc. 2017-18


This is a command-line encoding and streaming tool using the libobs library.

CMake build files are provided. Only currently tested build environment is Linux (Ubuntu 14.04).

Prebuilt libobs static library binaries are provided in libobs/lib.
